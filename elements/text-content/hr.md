## Linha Horizontal
Usamos o elemento `<hr>` para criar uma linha horizontal.

Sintaxe:

```html
<hr>
```
Exemplo:  
Vamos usar a linha horizontal para separar dois parágrafos

```html
<p>Isso é um parágrafo</p>
<hr>
<p>Isso é um parágrafo</p>
```

Resultado:  
<p>Isso é um parágrafo</p>
<hr>
<p>Isso é um parágrafo</p>
