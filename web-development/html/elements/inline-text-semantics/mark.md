### Marcação

`<mark>` - É usado para fazer marcações parecidas com uma caneta marca texto.

```html
<p>Amanhã irei <mark>comprar a placa de vídeo</mark>, fiquei o ano inteiro juntado dinheiro.</p>
```

Resultado:

><p>Amanhã irei <mark>comprar a placa de vídeo</mark>, fiquei o ano inteiro juntado dinheiro.</p>
