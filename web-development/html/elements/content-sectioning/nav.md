## Navegação

O elemento `<nav>` é usado para representar uma área de navegação, onde irá conter os principais links de navegação da página.

```html
<nav>
    <ul>
        <li><a href="#">Home</a></li>
        <li><a href="#">Sobre</a></li>
        <li><a href="#">Contato</a></li>
    </ul>
</nav>
```
