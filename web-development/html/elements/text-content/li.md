## Item da Lista

`<li>` - É usado para criar um item de uma lista.

O elemento `<li>` deve estar contido em um elemento pai: uma lista ordenada `<ol>`, uma lista desordenada `<ul>`, ou `<menu>` e representa um único item dessa lista.

Sintaxe:

```html
<li></li>
```

Exemplo:

```html
<ul>
  <li>pão</li>
  <li>Manteiga</li>
  <li>Café</li>
</ul>
```

Resultado:

<ul>
  <li>pão</li>
  <li>Manteiga</li>
  <li>Café</li>
</ul>
