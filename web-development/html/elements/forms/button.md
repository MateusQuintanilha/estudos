# Button

O elemento `<button>` representa um botão clicável. Podemos colocar conteúdo como texto ou imagem.
* Sempre especifique o atributo type.

Atributos:
* type
    - submit, reset, button
* autofocus
* disabled
* name
* value
* form

```html
<form id="form-name">
    <input type="text" name="name">
</form>

<button type="button"> Botão</button>
<button type="reset" form="forme-name"> Limpar</button>
<button type="submit" form="forme-name"> Enviar</button>
```
