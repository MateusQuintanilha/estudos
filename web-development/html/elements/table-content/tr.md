# Definindo as linhas de uma tabela
O elemento `<tr>` é responsável por criar as linhas da tabela

```html
<table>
    <tr>
        <th>A</th>
        <th>B</th>
        <th>C</th>
    <tr>
    <tr>
        <td>A1</td>
        <td>B1</td>
        <td>C1</td>
    <tr>
    <tr>
        <td>A2</td>
        <td>B2</td>
        <td>C2</td>
    </tr>
    <tr>
        <td>A3</td>
        <td>B3</td>
        <td>C3</td>
    </tr>
</table>
```
