# Combinador Lista

O combinador de lista é representado pela **virgula** `,`, esse combinador serve para agrupar vários seletores em uma única regra.

Sintaxe:

```css
seletor1, seletor2 {

}
```

Exemplo:

Vamos alterar a cor do texto para a cor vermelha nos elementos `<h1>`, `<h2>`, `<h3>`, `<p>`.

```css
h1, h2, h3, p {
  color: red;
}
```
