# Grado / Gradians (grad)

Representa o ângulo em grado que é representado por (grad), onde se divide um círculo em 400 partes e cada uma das 400 divisões do circulo corresponde a 1grad

```css
400grad
```

* 30º = 33,3333grad
* 45° = 50grad
* 60° = 66,6667grad
* 90° = 100grad
* 180° = 200grad
* 270° = 300grad
* 360° = 400grad
