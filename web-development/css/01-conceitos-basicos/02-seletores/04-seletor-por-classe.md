# Seletor por Classe

Seleciona todos os elementos baseado no valor do atributo `class=""` fornecido. Todo seletor por classe ser√° iniciado com um ***ponto*** `.`.

Sintaxe:

```css
.nome-da-classe {
  propriedade: valor;
}
```

Exemplo:

Vamos selecionar todos os elementos que possuem o atributo `class` com valor `"fundo-amarelo"`

```css
.fundo-amarelo {
  background-color: yellow;
}
```
