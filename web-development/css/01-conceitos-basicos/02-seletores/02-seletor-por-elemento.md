# Seletor de Elemento

Podemos usar o nome de qualquer elemento HTML para aplicar as regras css, mas as regras ser√£o aplicadas a todos os elementos que corresponderem ao nome fornecido.

Sintaxe:

```css
nome-do-elemento {
  propriedade: valor;
}
```

Exemplo:

Vamos selecionar todos os elementos`<p>` e troca a cor da fonte para azul.

```css
p {
  color: blue;
}
```
